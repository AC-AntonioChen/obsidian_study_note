
## 1 Redis 基础
### 1.1 Redis 有什么作用?为什么要用 Redis/为什么要用缓存？
Redis是流行的内存型数据库，具有高速缓存的特性。支持多种数据结构，例如字符串，列表，哈希map，集合，有序集合等，
使用redis 的原因主要有
（1）访问速度更快：redis是内存型数据库，内存访问速度比磁盘快很多，将高频数据放到redis中，相比磁盘可以提升几十倍的读取速度
（2）高并发：通过将热点数据从数据库移到缓存中去。可以减少数据库的压力。mysql数据库 一般云服务商4c8g实例的qps也就几千这个数量级，使用redis的话，单机redis就可以达到几万数量级的并发访问了
（3）功能全面：redis除了可以用于缓存，还可以用于分布式锁，限流，延时队列，甚至是消息队列
### 1.2 Redis 除了做缓存，还能做什么？
可以用于实现分布式锁、限流、延时队列，甚至是消息队列
