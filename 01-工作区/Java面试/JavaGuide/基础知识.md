# 一、基础
### 1、Java 中的几种基本数据类型了解么？

Java 中有 8 种基本数据类型，分别为：
- 6 种数字类型：
    - 4 种整数型：`byte`、`short`、`int`、`long`
    - 2 种浮点型：`float`、`double`
- 1 种字符类型：`char`
- 1 种布尔型：`boolean`。

这 8 种基本数据类型的默认值以及所占空间的大小如下：

|基本类型|位数|字节|默认值|取值范围|
|---|---|---|---|---|
|`byte`|8|1|0|-128 ~ 127|
|`short`|16|2|0|-32768（-2^15） ~ 32767（2^15 - 1）|
|`int`|32|4|0|-2147483648 ~ 2147483647|
|`long`|64|8|0L|-9223372036854775808（-2^63） ~ 9223372036854775807（2^63 -1）|
|`char`|16|2|'u0000'|0 ~ 65535（2^16 - 1）|
|`float`|32|4|0f|1.4E-45 ~ 3.4028235E38|
|`double`|64|8|0d|4.9E-324 ~ 1.7976931348623157E308|
|`boolean`|1||false|true、false|


### 2、JVM内存模型


| 部分                              | 线程共享/私有 | 作用                         | 示例                                    |
| ------------------------------- | ------- | -------------------------- | ------------------------------------- |
| 堆（Heap）                         | 线程共享    | 存储所有的对象实例以及数组              | `new Object()`                        |
| 方法区（Method Area）                | 线程共享    | 存储类信息、常量、静态变量、即时编译器编译后的代码  | `class A { static int x = 10; }`      |
| 方法栈（JVM Stack）                  | 线程私有    | 存储局部变量表、操作栈、动态链接、方法出口，对象指针 | `int a = 1; Object b = new Object();` |
| 本地方法栈（Native Method Stack）      | 线程私有    | 为虚拟机使用到的Native 方法服务        | `System.arraycopy()`                  |
| 程序计数器（Program Counter Register） | 线程私有    | 记录当前线程所执行的字节码的行号指示器        | `pc = 10`                             |
